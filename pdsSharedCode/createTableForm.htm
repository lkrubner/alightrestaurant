You can use this form to create a database table

An auto-incrementing primary key field called "id" is added automatically, so don't add it yourself.

If you add a field called "private" then the user will have power to control whether something is private. A value of "t" for true will make an entry private.

If you add a field called "upload_file" then the forms created will allow for a file to be uploaded file (like an image or pdf)

If you want your records to have a headlines or a caption, use the name "title" for the field.
